Please excuse my formatting, I'm only just learning my CSS.

First, a write up of this algorithm for the analog amongst us.

  • Pack laptop and lunch into bag
  • Pickup keys from bowl
  • Walk downstairs
  • Pump tyres up on bike
  • Connect bag to bike
  • Unlock bike
  • Put on helmet
  • Get on bike
  • Ride bike to university, stop for any intersections and check for cars.
  • Lock bike to bike racks.
  • Take bag off bike
  • Pack laptop and lunch into bag
  • Pack laptop and lunch into bag
  • Pack laptop and lunch into bag

Some Pseudocode

Now, here is that algorithm converted into some sort of pseudocode.

START
IF bag_packed == FALSE:
DO pack bag
bag_packed = TRUE

IF cyclist_keys == FALSE:
DO put keys into pocket

IF bicycle_locked == TRUE:
DO WHILE bicycle_locked == TRUE:
Test cyclist_keys in bicycle_lock until bicycle_locked == FALSE

IF helmet_on_head == FALSE:
DO Put helmet on head

IF bag_on_bike == FALSE:
DO Add bag to bike

WHILE tyre_pressure < 80:
DO tyre_pressure += 1

WHILE at_destination == FALSE:
cycle_to_destination = TRUE

WHILE cycle_to_destination == TRUE:
DO Pedal for 10 seconds
DO check_for_intersection
IF check_for_intersection == TRUE:
DO is_traffic
WHILE is_traffic == TRUE:
wait for 5 seconds
DO is_traffic
DO at_destination
IF at_destination == TRUE:
cycle_to_destination = FALSE

DO Lock bicycle to bike rack
bicycle_locked = TRUE

DO remove helmet
helmet_on_head = FALSE

DO take bag off bike
bag_on_bike = FALSE

FINISH
